Blogging Platform

Project Overview

A complete blogging platform with Admin Panel, user authentication, post management, categories and responsive design. A full featured bloggong platform that allows users to read, search, and interact with blog posts, while administrators can manage content through a secure admin dashboard

Features

  • User Registration & Login
  • Admin Dashboard
  • Create / Edit / Delete Posts
  • Category Management
  • Responsive Design

Technologies Used

  • HTML
  • CSS3
  • JavaScript
  • React.js
  • Mongodb
