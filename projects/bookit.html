bookit

Project Overview

Developed a responsive hotel and travel booking website that allows users to search destinationss, explore hotels, view places and make booking through an intuitive interface. The platform focuses on providing a seamless booking experience with a modern UI/UX design.

Features

  • Destination search functionality
  • online booking interface
  • Price and rating display
  • User friendly navigation
  • Responsive Design for all devices
  • contact and support section
  • Interactive UI components

Technologies Used

  • HTM5L
  • CSS3
  • JavaScript
  • Tailwind CSS
  • Rest API
  • React
  • Mongodb
