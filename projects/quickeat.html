Quickeat landing page

Project Overview

Developed a modern and responsive food delivery landing page that showcases restaurant services, menu highlights, special offers, and many more. The project focuses on attractive UI/UX design and mobile responsiveness to improve user engagement.

Features

  • Popular restaurants showcase
  • featured food items section
  • customers reviews
  • Special offers and discount section
  • Responsive Design
  • Smooth scrolling navigation
  • fast loading optimized interface

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • React.js
  • Tailwind CSS
